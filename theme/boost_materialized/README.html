boost_materialized theme - Installation steps

  • Please unzip the downloaded theme zip file, you will get folder 'boost_materialized'
  • Copy folder 'boost_materialized' and put into theme folder of your moodle system
  • Next login as Site administrator
  • Go to 'Site administration' -> 'Notifications' , here on 'Plugins check' page you will find both the items.
  • Click the "Upgrade Moodle database now" button displayed on bottom of the page
  • You will get success message once the plugins installed successfully.
  • By clicking "Continue" button on success page , you will get options to change the boost_materialized theme settings.
  • You can modify the settings later too :-)

Steps to set boost_materialized theme as default theme for your moodle system

  • Login as site administrator
  • Go to Site administration -> Appearance -> Themes -> Theme selector
  • Click the "Change theme" button for device type as "Default"
  • It will list all the available themes for "Default device"
  • Then click "Use theme" button on "boost_materialized theme"
  • Next click the "Continue" button ,thats it.
  • Cheers,you have done it !!!
